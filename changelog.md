# Changelog

## [1.0.0] - 2026-03-13

### Added
- Credential profile management (domain and local accounts)
- AES-256 encrypted password storage
- Multi-device selection with search and filtering
- Batch interactive console session creation on Windows 10/11
- Dual session creation methods (cmdkey+mstsc with tscon fallback, scheduled task fallback)
- Connection history/logging
- French localized interface
- Online/offline device status display
- Select all / deselect all for visible devices
